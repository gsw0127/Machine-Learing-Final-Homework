课程大作业 README

📚 课程大作业 - 项目说明

本仓库用于存放机器学习课程大作业的核心代码、训练模型、实验结果及相关数据集处理文件，基于 YOLOv8 模型完成相关任务开发与测试。

所有代码、实验日志及训练结果已完整归档，可直接下载运行或查阅验证。

📂 仓库目录结构说明

以下为仓库核心目录及文件的详细说明，清晰标注各文件夹用途及包含内容：

- 23281147pbh_Weight/ - 模型权重文件目录
存放本人训练得到的 YOLOv8 模型权重文件，含 yolov8n 基础模型权重、两个 VisDrone 数据集训练后的权重结果。

- COCO2017/ - COCO2017 数据集处理目录
存放 COCO2017 数据集的预处理脚本、数据划分配置、标签转换工具等相关文件，用于数据集的加载与适配模型训练。

- Datasets/ - VisDrone2019 数据集处理目录（已删除）
原用于存放 VisDrone2019 数据集的处理脚本及相关配置文件，目前已删除，保留目录用于记录项目历史结构。

- Log/ - 训练日志目录
存放模型训练过程中的完整日志文件，包含训练参数、损失曲线数据、评估指标变化等信息，可用于追溯训练过程及问题排查。

- RES/ - 训练/测试结果目录
存放模型训练、测试过程中生成的预测结果文件，用于结果分析与验证。

- RES_Photo/ - 结果截图目录
存放训练过程中的损失曲线截图、测试集预测效果截图等可视化文件，可直接证明本人真实完成本次大作业。

- test/ & train/ 脚本文件 - 核心执行脚本
包含模型训练（train 相关脚本）、测试推理（test 相关脚本）的完整代码，配置好环境后可直接运行启动训练或测试任务。
